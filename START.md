# 快速启动指南

## 系统要求
- Node.js 18+ (必需)
- Blender 3.6+ (可选，用于 AI 绘画)

## 启动方式

### macOS
1. 双击 `start.command` 文件
2. 首次运行会自动安装依赖
3. 浏览器自动打开 http://localhost:3001

### Windows
1. 双击 `start.bat` 文件
2. 首次运行会自动安装依赖
3. 浏览器自动打开 http://localhost:3001

### 命令行方式
```bash
npm install
node server.js
```

## 功能说明

### 3D 拆解视图
- 拖动滑块查看拆解步骤
- 鼠标悬停查看部件信息
- 点击部件高亮显示

### AI 绘画 (需要 Blender)
- 输入提示词生成 3D 模型
- 支持乐高风格、超级英雄、车辆等
- 自动生成拆解动画

### 支持的提示词示例
- `Quest 3` - VR 头显
- `乐高人仔` - 乐高人物
- `蝙蝠侠` - 超级英雄
- `乐高跑车` - 乐高车辆
- `乐高房子` - 乐高建筑

## 停止服务
- 在终端窗口按 `Ctrl+C`
- 或直接关闭终端窗口
