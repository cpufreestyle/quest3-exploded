## 🚀 新增功能亮点

### 1. 智能爆炸算法
```javascript
// 自定义模型自动计算爆炸方向
const center = new THREE.Vector3();
box.getCenter(center);
partPositions.forEach((pos, idx) => {
  const direction = pos.clone().sub(center).normalize();
  targetPositions[idx] = pos.clone().add(direction.multiplyScalar(distance));
});
```
