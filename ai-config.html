图片转 3D（提供商配置）

把上传图片重建为带贴图的 GLB。选择一个提供商：本地 TripoSR（离线真重建，无需 API）或 Replicate 云端 API。

本地模式由 server.js 调用 scripts/triposr_infer.py（TripoSR 真重建）离线生成带体积的真·GLB。首次需联网下载模型权重，请先运行 bash scripts/setup_triposr.sh。

越大越精细也越慢（建议 256–384，CPU 首跑较慢）

勾选后用 rembg 自动抠图，效果更干净但依赖更重

从 Replicate API Tokens 获取

格式：owner/name。例如 triposr、hunyuan3d-2 等

留空优先用上面的 owner/name；填入具体版本哈希则按版本调用
