# 🤖 AI 智能爆炸调整功能

## 功能说明

智能爆炸距离调整会根据以下因素自动计算最佳爆炸距离：

### 1. **模型尺寸**
- 爆炸距离 = 模型最大尺寸 × 40%
- 确保爆炸效果与模型大小成比例

### 2. **相机位置**
- 计算模型到相机的距离
- 避免爆炸后飞出视野

### 3. **视野角度（FOV）**
- 基于相机 FOV 计算最大可见距离
- 留 40% 边距防止部件靠近屏幕边缘

### 4. **爆炸方向**
- 如果爆炸方向朝向相机，自动减小距离
- 确保不会飞出屏幕

## 计算逻辑

```javascript
// 1. 基础距离：模型尺寸的 40%
suggestedDist = maxDim × 0.4

// 2. 计算视野边界
maxVisibleDist = distToCamera × tan(FOV/2) × 0.6

// 3. 考虑角度
if (爆炸方向朝向相机) {
  angleFactor = 0.5 ~ 1.0 (减小距离)
}

// 4. 取最小值，确保不飞出屏幕
suggestedDist = min(suggestedDist, maxVisibleDist × angleFactor)

// 5. 确保最小可见性
suggestedDist = max(suggestedDist, 1.0)
```
