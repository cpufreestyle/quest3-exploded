🔄 清除浏览器缓存

遇到 loader.parse is not a function 错误？
这通常是因为浏览器缓存了旧版本的 JavaScript 文件。
📋 手动清除缓存步骤：
  1. 打开浏览器开发者工具（F12 或 Cmd+Option+I）
  2. 右键点击刷新按钮 🔄
  3. 选择 "清空缓存并硬性重新加载"
  4. 等待页面重新加载完成
💡 快捷键：
• Mac: Cmd + Shift + R
• Windows/Linux: Ctrl + Shift + R
• 或者: Cmd + Option + E (Mac) 清空缓存
✅ 缓存已清除！正在重新加载...
⚠️ 无法自动清除缓存，请使用手动方法
